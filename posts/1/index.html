记录此博客的搭建和使用

发表于 2019-03-28 | 更新于 2019-03-29 | 分类于 杂

为了不忘掉这博客怎么用，我还是记录一下为妙

所有前面带git或者hexo的若没有特殊说明，都是在E:\Blog中右键后点击Git Bash Here后输入的，必须要在前面出现”$”符号时输入，要求输入账号密码时要自己看

软件

  • nodejs
  • git for windows​
  • Typora​

搭建步骤

我也不知道他怎么搭的，如果知道并且能听懂的话我会写一下

使用方法

  • 用Typora打开E:\Blog\source_posts*.md，然后直接编辑
  • 编辑时第4行的”tags:”表示标签（可有多个），”categories:”表示分类（只能有一个），标签和分类对应的名称直接写在对应的栏之后就行。如果有多个标签，就要换行，输入” - 标签名”（前面是两个空格）
  • ““这句话表示阅读全文
  • 编辑完保存后输入hexo s的话可以在http://localhost:4000 中查看（等它两行加载出来就行了），输入hexo clean，加载完后输入hexo g -d后可以在https://cnt27.github.io/ 上查看
  • 用户名等信息均可以在E:\Blog\_config.yml中修改

上传到github上让自己在另一台电脑上也能使用

上传

  • git add .
  • git commit -m “update”（sxyugao说update替换成什么都行，我反正不清楚）
  • git push

另一台电脑上下载

  • 打开https://github.com/cnt27/Blog 点击绿色的”Clone or download”，然后点击在地址右侧的那个符号
  • 在E:\中输入git clone ，粘贴在第一步中复制的地址后enter
  • 下面照常在E:\Blog中操作
  • npm i
  • git pull
